# Fashion MNIST Classifier

**Clothing Item Classification using Artificial Neural Network (ANN)**

A complete deep learning project to classify 10 different clothing items from 28x28 grayscale images using Keras and TensorFlow.

![Fashion MNIST](https://github.com/user/fashion-mnist-classifier/blob/main/sample.png)

---

## 📋 Project Overview

- **Title**: Fashion MNIST Classifier
- **Description**: Clothing item classifier built using Fashion MNIST dataset
- **Model**: Artificial Neural Network (ANN) with Dense and Dropout layers
- **Framework**: TensorFlow & Keras
- **Accuracy**: ~90.5% on test set

---

- Artificial Neural Networks (ANN)
- Dense (Fully Connected) Layers
